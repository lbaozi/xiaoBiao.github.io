本文只是对官方文档的翻译整理

一、 安装

  1. 安装NodeJS,可以选择从官网下载安装也可以选择使用nvm安装
  2. 安装Git,Window下载安装即可,MAC可以使用Homebrew安装
  3. 安装Cordova CLI

二、 创建项目

创建项目使用 cordova create 命令: 

cordova create hello com.example.hello HelloWorld

hello为项目路径,该路径必须是不存在的路径
com.example.hello为项目id,建议使用反向域名确保唯一性,为可选参数,默认值为io.cordova.hellocordova,可以通过config.xml进行修改
HelloWorld为项目名称,可选参数,可以通过config.xml修改 

cd hello

生成的项目中还需要添加平台支持,针对不同的系统添加相应的代码,使用cordova platform add命令,按照需要进行添加即可

cordova platform add ios
cordova platform add android
cordova platform add amazon-fireos
cordova platform add blackberry10
cordova platform add firefoxos

可以使用cordova platform ls查看已经添加的平台
使用cordova platform rm或cordova platform rmove移除平台

cordova prepare命令会将修改的文件复制到所有的平台文件夹中准备进行编译,例如config.xml、www文件夹中的修改等,也可以在命令后添加平台代码,只更新指定平台的代码如cordova prepare ios

cordova compile命令会编译所有的平台代码,也可使用各平台自己的开发工具进行编译.例如ios平台可以通过xcode打开platform/ios中的项目进行编译,该命令同样可以指定平台

cordova build命令等价于以下两条命令会

cordova prepare
cordova compile

创建项目并添加平台后即可通过各自的开发工具进行开发
ios平台需要MACOS环境,并安装xcode,可能还需要苹果的开发者帐号

三、 配置文件

cordova的配置文件目录为项目根目录的config.xml
ios配置文件目录为 platforms/ios/项目名称/config.xml
Android配置文件目录为 platforms/android/res/xml/config.xml

全平台属性

Fullscreen为全平台属性,支持所有设备设置是否隐藏状态条,默认值为fasle(不过我设置为true了也没啥用可能设置的有问题吧) 

<preference name="Fullscreen" value="true" />

多平台属性

以下属性支持多个平台,单并不不是所有平台

DisallowOverscroll取消webview的回弹效果,支持iOS和Android,默认值为false

<preference name="DisallowOverscroll" value="true"/>

BackgroundColor设置APP的背景颜色,支持Android和BlackBerry

<preference name="BackgroundColor" value="0xff0000ff"/>

HideKeyboardFormAccessoryBar键盘上方的表单工具条,用于切换表单上一项下一项,支持iOS,BlackBerry,默认值为false

<preference name="HideKeyboardFormAccessoryBar" value="true"/>

Orientation锁定APP的方向,支持Android, iOS, WP8, Amazon Fire OS,Firefox OS

<preference name="Orientation" value="landscape" />

feature标签

使用命令行添加插件时会自动加入该标签,该标签只存在与各平台的confg.xml中

platform标签

用来指定特定平台的属性设置,例:

<platform name="android">
    <preference name="Fullscreen" value="true" />
</platform>

四、 安装插件

安装插件使用cordova plugin add命令,旧版插件库地址为:http://plugins.cordova.io/#/,新版已经迁移至npm,在npm中搜索ecosystem:cordova,旧版本的插件也已更名,例如Device插件旧版添加命令为

cordova plugin add org.apache.cordova.device

新版本为:

cordova plugin add cordova-plugin-device

执行添加插件命令会为该项目下所有平台添加相应插件
